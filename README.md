# Artefatos do estudo

Este repositório contém os artefatos anonimizados utilizados no estudo sobre feedback formativo com LLMs em ambientes de programação baseados em juízes automáticos.

Os arquivos foram preparados para permitir a reprodução das métricas reportadas no artigo, preservando apenas as variáveis necessárias para as análises pedagógica e operacional. Por razões de privacidade, minimização de dados e proteção dos materiais didáticos, foram removidos identificadores pessoais, identificadores de turma, datas e horários precisos, enunciados integrais, códigos completos de estudantes, códigos de referência, textos completos de feedback, cargas JSON brutas, hashes internos e logs completos da aplicação.
